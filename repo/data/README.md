# Data artefacts

This directory holds the reproducible CSV exports produced by the notebook. The files are git-ignored because they are regenerated on every run; this README documents what each one contains.

| File | Description |
|---|---|
| `sampled_synthetic_american_option_data.csv` | The 4,998-row stratified synthetic sample used for modelling. |
| `SPY_option_chain_raw.csv` | Raw SPY option chain as downloaded from Yahoo Finance. |
| `SPY_option_chain_clean.csv` | Cleaned SPY snapshot after market-quality filters. |
| `SPY_iv_model_results.csv` | Implied-volatility model comparison table. |
| `SPY_spread_model_results.csv` | Bid-ask spread model comparison table. |
| `SPY_iv_diagnostics.csv` | Actual vs predicted IV with residuals for the best model. |
| `SPY_spread_diagnostics.csv` | Actual vs predicted spread with residuals for the best model. |

To regenerate them, run the export cells at the end of
`notebooks/H9MLFF_CA2_option_pricing.ipynb`.

> Note: the synthetic source dataset is generated by the module convenor's
> 15,000-step binomial routine and is not redistributed here. Point the
> notebook's loader at your local copy to reproduce the synthetic track.
